Bienvenue sur PrideGPT

PrideGPT est une version améliorée de ChatGPT, capable de faire bien plus que répondre à des questions textuelles. Voici ce que PrideGPT peut faire pour vous :

  • Génération d'images avec Dall-E
  • Réponses vocalisées
  • Et bien plus encore...

Choisir le modèle de GPT :

  • GPT 3.5 Turbo
  • GPT 4

Choisir le modèle DALL-E :

  • DALL-E 2
  • DALL-E 3

Choisir la voix :

  • Alloy
  • Echo
  • Fable
  • Onyx
  • Nova
  • Shimmer

Choisir le langage de programmation :

  • Python
  • JavaScript
  • Java
  • C#
  • C++

Choisir le mode de réponse

  • Text
  • Speech

Utilisez les commandes spéciales pour accéder à des fonctionnalités avancées :

  • /image [Text] pour générer des images avec Dall-E
  • /speech [Text] pour obtenir une réponse vocalisée. Il faut choisir au préalable le mode "Speech" comme mode de réponse
  • /translate [Langue] [Texte] traduire un texte
  • /code [Text] pour obtenir un bloc de code. Il faudra choisir le type de langage de programmation
  • /transcribe Permet de retranscrire du texte à partir d'un audio. Il faudra joindre un audio
  • Bouton vocal Qui permet de faire des requetes avec le son de votre voix. Il faudra choisir le type de réponse, entre Speech et text
